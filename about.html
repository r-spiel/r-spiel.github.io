About Me

Hello! My name is Richelle, and I am a work in progress.
Some of my friends call me a “hype person” because my enthusiasm can be so infectious. I am currently super enthusiastic about learning to code! I have loved teaching myself HTML and CSS in my free time from books and online resources (check my projects page for details). I am now diving into learning Ruby while tinkering with my website a bit here and there. I am currently working full time and hope to join a full time coding bootcamp soon.

I’ve lived in Seattle for the last 13 years, attending the University of Washington where I got a Bachelor’s in Economics and History in 2011. I have worked in teaching, finance and sales (check out my LinkedIn for more details) - but what I have learned from them all is that I love learning, building, problem solving and working collaboratively with people. Coding fills those areas and as a bonus I find it really fun!

My favorite part when I am coding is when I realize that I am on the cusp of solving a problem. These moments give me momentum and build my anticipation for solving whatever problem I have encountered. I have seen with editing my own code that solving one problem can sometimes lead to unintended consequences, but this really drives me to learn more about the relationship between all the code that I am using so that I can make any project I am working on look and work just the way I want it to.

I built this site by writing the code in HTML & CSS (in the text editor VS Code), and hosting in on GitHub Pages. You can read about some of the articles that helped me build this site by going to this link or by using the "Projects" drop down menu above and selecting HTML & CSS. This website is not perfect, and is a constant work in progress, just like me! Things should change on this site from time to time as I fine tune the code and design of the site and as I build more projects to write about! Hopefully as you check back you will begin to see some changes and improvements.

One of my passions is listening to podcasts, audio books and reading (my daily commute gives me a good chance to get some audio listening in).
