My first coding project, in February 2019 was making a game in Scratch. I was working on the EdX course: Harvard CS50 Intro to Computer Science, where the Week 0 assignment is to make a game in Scratch within certain parameters. I found this to be a very fun and educational experience! After listening to the Week 0 lecture, I started out in Scratch creating one of the Level one games from the book Code Your Own Games!: 20 Games to Create with Scratch, this helped me learn the basics of scratch before moving on to my own creation. I was able to create the game that I submitted for this assignment (linked below) in one full Saturday after many hours of work. I loved the process and knew I wanted to learn more about coding! This assignment also introduced me to core concepts like loops, variables and conditions.
