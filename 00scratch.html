My first coding project, in February 2019, was making a game in Scratch. I was working on the EdX course: Harvard's CS50 Intro to Computer Science, where the Week 0 assignment is to make a game in Scratch within certain parameters. I found this to be a very fun and educational experience! After listening to the Week 0 lecture, I started out in Scratch creating one of the Level One games from the book Code Your Own Games!: 20 Games to Create with Scratch. This helped me learn the basics of Scratch before moving on to my own creation. I was able to create the game that I submitted for this assignment (linked below) in one full Saturday after many hours of work. I loved the process and knew I wanted to learn more about coding! This assignment also introduced me to core concepts like loops, variables and conditions. Although I only completed Week 0 of the course before moving on to learn HTML & CSS in order to code this website, I am still very glad that I did it! It was a great introduction and a lot of fun.

Play the game embedded below, or click here to play it in Scractch where you can also see the code that makes it work. All the graphics and images are of course part of the MIT Scratch library which makes it such a fun coding program to play with. **HINT** - the keys can sometimes hide behind the counters! Not sure why, but it seems to happen often. Sound ON for the best experience! ;)
